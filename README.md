# Tailwind & JS Login Practice

Um projeto de página de login moderna
desenvolvida para praticar a intergração
entre **Tailwind CSS** e a lógica de validação
com **JavaScript**

## 🚀 Tecnologias Utilizadas
* **HTML5** - Estrutura semântica
* **Tailwind CSS** - Estilização moderna e responsiva
* **JavaScript** - Lógica de validação de formulários e manipulação de DOM
